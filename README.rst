Prerequisites
-------------

You'll need the following Python modules installed:
 - django (1.4+)
 - python-django-auth-openid


Configuration and Usage
-----------------------

Copy local_settings.py.sample to local_settings.py and change
settings there.

Create empty database:
./manage.py syncdb
